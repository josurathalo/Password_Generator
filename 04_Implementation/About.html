เกี่ยวกับ

เครื่องมือนี้ช่วยสร้างรหัสผ่านสุ่มที่ปลอดภัย โดยผู้ใช้กำหนดความยาวและเลือกประเภทอักขระได้

Since 2022

ประวัติความเป็นมา

ทีมเริ่มต้นโครงการจากความต้องการสร้างเครื่องมือออนไลน์ที่ช่วยให้ผู้ใช้ทั่วไปสามารถสร้างรหัสผ่านที่ปลอดภัยได้อย่างรวดเร็ว โดยไม่ต้องติดตั้งซอฟต์แวร์เพิ่มเติม เราเน้นใช้งานบนเบราว์เซอร์ด้วย Web Crypto API เพื่อให้การสุ่มมีความปลอดภัยมากกว่าการใช้ Math.random

พันธกิจ (Mission)

มุ่งมั่นส่งมอบเครื่องมือที่ปลอดภัย เข้าถึงได้ง่าย และโปร่งใส เพื่อช่วยให้ผู้ใช้ปกป้องบัญชีและข้อมูลส่วนตัวอย่างมีประสิทธิภาพ

วิธีการทำงาน

  • ใช้ crypto.getRandomValues เพื่อสุ่มตัวอักษรอย่างปลอดภัย
  • รับประกันว่าถ้าเลือกหลายประเภทจะมีตัวอักขระจากแต่ละประเภทอย่างน้อยหนึ่งตัว
  • ไม่เก็บรหัสผ่านที่ผู้ใช้สร้างในเซิร์ฟเวอร์ (ถ้าไม่ได้ตั้งค่าเพิ่มเติม)

ความปลอดภัย

เราเน้นการใช้ API ของเบราว์เซอร์และหลักปฏิบัติที่ดี: ไม่ส่งรหัสผ่านที่สร้างไปยังเซิร์ฟเวอร์โดยไม่จำเป็น ผู้ใช้ควรตรวจสอบว่ากำลังใช้งานผ่าน HTTPS และเก็บรหัสผ่านในตัวจัดการรหัสผ่านที่เชื่อถือได้

แผนพัฒนา (Roadmap)

  1. เพิ่มระบบบัญชีผู้ใช้และซิงค์แบบเข้ารหัส (เลือกเปิดใช้)
  2. เชื่อมต่อการชำระเงินแบบปลอดภัย (Stripe) สำหรับแผน Pro
  3. เสนอ API สำหรับองค์กรและ SSO

ทีมงาน

ทีมประกอบด้วยนักพัฒนาเว็บความปลอดภัยและ UI/UX ที่ทำงานร่วมกันเพื่อลดความซับซ้อนของ UX และเพิ่มความปลอดภัยพื้นฐาน
